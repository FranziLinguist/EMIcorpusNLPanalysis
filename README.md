# EMIcorpusNLPanalysis

NLP analysis pipeline for an English-medium instruction (EMI) academic corpus. Compares linguistic and rhetorical features across disciplines using a 14-stage pipeline covering tokenization, morphology, verb analysis, agency, nominalization, cohesion, academic formulas, frequency, TF-IDF keywords, topic modeling, dependency parsing, and cross-discipline similarity.
